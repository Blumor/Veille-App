EXEMPLES DE PLANIFICATION (cron) — vraie automatisation
========================================================

Le générateur CLI (src/index.js) produit un rapport et l'enregistre, sans
intervention. Couplé à cron (Linux/macOS) ou au Planificateur de tâches
(Windows), il publie tout seul — y compris quand l'app n'est pas ouverte.

Prérequis : `npm install` fait. AUCUNE clé API requise — la génération est
gratuite (RSS + NVD + CISA KEV).
Remplace /chemin/vers/veille-cyber par le chemin absolu réel.


--- crontab -e (Linux / macOS) ---

# Rapport quotidien tous les jours à 6h30
30 6 * * *  cd /chemin/vers/veille-cyber && /usr/bin/node src/index.js daily   >> data/cron.log 2>&1

# Rapport hebdomadaire le dimanche à 7h00
0 7 * * 0   cd /chemin/vers/veille-cyber && /usr/bin/node src/index.js weekly  >> data/cron.log 2>&1

# Rapport mensuel le 1er du mois à 7h30
30 7 1 * *  cd /chemin/vers/veille-cyber && /usr/bin/node src/index.js monthly >> data/cron.log 2>&1


--- Astuce ---

Trouve le chemin de node avec :  which node
Le serveur web (npm start) peut tourner en continu à côté : il lira
simplement les rapports déposés par cron dans data/reports/.
